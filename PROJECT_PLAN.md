# FIFA 全局项目规划

> 说明：当前仓库里没有独立的历史规划文档，本文件中的“已完成规划”是根据现有代码实现回填整理，用于统一记录项目全局方向与当前进度。

## 1. 项目目标

- 打造一个面向 Web3 场景的 FIFA 前端控制台/工作台。
- 基于 React 19 + TypeScript + Vite 建立可持续扩展的业务前端骨架。
- 优先把路由、数据请求、钱包连接、国际化、状态管理、图表能力等基础设施打通，再逐步进入业务模块开发。

## 2. 当前项目定位

- 当前阶段：前端业务脚手架已完成，已具备继续承接真实业务模块的条件。
- 当前形态：以 Dashboard + Playground 为核心的管理台原型。
- 当前网络方向：已接入 BNB Smart Chain 作为默认 EVM 网络。
